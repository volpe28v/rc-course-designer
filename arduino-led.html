LED配線図

13 (LED)
GND
2 (SW)

配線手順:

  • LEDのアノード（長い足）をピン13に接続
  • LEDのカソード（短い足）を抵抗経由でGNDに接続
  • スイッチの一端をピン2に接続
  • スイッチのもう一端をGNDに接続
// LEDとスイッチの制御プログラム const int LED_PIN = 13; // LED接続ピン const int SWITCH_PIN = 2; // スイッチ接続ピン void setup() { pinMode(LED_PIN, OUTPUT); // LEDピンを出力に設定 pinMode(SWITCH_PIN, INPUT); // スイッチピンを入力に設定 } void loop() { // スイッチの状態を読み取り、LEDを制御 if (digitalRead(SWITCH_PIN) == HIGH) { digitalWrite(LED_PIN, HIGH); // LED点灯 } else { digitalWrite(LED_PIN, LOW); // LED消灯 } }
